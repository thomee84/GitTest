C# Metódusok

A metódus a kód egy blokkja, ami csak akkor fut, amikor meghívják. Tudunk hozzáadni adatokat a metódusokhoz, amelyeket paramétereknek hívunk. A metódusokat bizonyos műveletek végrehajtására használják (függvények).

A metódusokat a kód újrafelhasználása miatt használjuk. "Definiáld egyszer, használt többször".

A metódust a nevével definiáljuk, majd () végződik. A C# biztosít néhány előre definiált metódust, amik már ismertek, pl. a Main() metódus, de saját metódus is létrehozható:

class Program
{
static void MyMethod()
{
// code to be executed
}
}

  • A MyMethod() a metódus neve.
  • A "static" azt jelenti, hogy a metódus a Program osztályába tartozik és nem a Program osztály egy objektuma.
  • A "void" azt jelenti, hogy ennek a metódusnak nincs visszatérési értéke.
A metódus meghívásához a neve és a (); kell:static void MyMethod()
{
Console.WriteLine("I just got executed!");
}

static void Main(string[] args)
{
MyMethod();
}

// Outputs "I just got executed!"

Egy metódus többször is meghívható:static void MyMethod()
{
Console.WriteLine("I just got executed!");
}

static void Main(string[] args)
{
MyMethod();
MyMethod();
MyMethod();
} // I just got executed! // I just got executed! // I just got executed!

Metódusok paraméterei

A paraméterek változókként léteznek a metódusok belsejében:static void MyMethod(string firstName)
{
Console.WriteLine(firstName + " Lastname");
}
static void Main(string[] args)
{
MyMethod("Liam");
MyMethod("Jenny");
MyMethod("Anja");
}
// Liam Lastname
// Jenny Lastname
// Anja Lastname

Alapértelmezett érték beállítása a paraméterre:static void MyMethod(string country = "Norway")
{
Console.WriteLine(country);
}
static void Main(string[] args)
{
MyMethod("Sweden");
MyMethod("India");
MyMethod();
MyMethod("USA");
}
// Sweden
// India
// Norway
// USA

Több paraméter használata:static void MyMethod(string fname, int age)
{
Console.WriteLine(fname + " is " + age);
}
static void Main(string[] args)
{
MyMethod("Liam", 5);
MyMethod("Jenny", 8);
MyMethod("Anja", 31);
}
// Liam is 5
// Jenny is 8
// Anja is 31

Visszatérési értékek:

A "void" kulcsszó azt jelzi, hogy a metódusnak nem rendelkezik visszatérési értékkel. Ha visszatérési értéke van, akkor primitív adattípust (például int vagy double) használható "void" helyett, és a visszatérési kulcsszót a metódus belsejében kell használni:

Visszatérési értékkel:static int MyMethod(int x)
{
return 5 + x;
}
static void Main(string[] args)
{
Console.WriteLine(MyMethod(3));
}
// Outputs 8 (5 + 3)

Visszatérési értékkel és több paraméterrel:static int MyMethod(int x, int y)
{
return x + y;
}
static void Main(string[] args)
{
Console.WriteLine(MyMethod(5, 3));
}
// Outputs 8 (5 + 3)
// Outputs 8 (5 + 3)

Változóban is tárolható az eredmény (ajánlott, mivel könnyebben olvasható és karbantartható).

Visszatérési értékel, több paraméterrel, változóban tárolva: static int MyMethod(int x, int y)
{
return x + y;
}
static void Main(string[] args)
{
int z = MyMethod(5, 3);
Console.WriteLine(z);
}
// Outputs 8 (5 + 3)

Elnevezett argumentumok:

static void MyMethod(string child1, string child2, string child3)
{
Console.WriteLine("The youngest child is: " + child3);
}
static void Main(string[] args)
{
MyMethod(child3: "John", child1: "Liam", child2: "Liam");
}
// The youngest child is: John

Az elnevezett argumentumok hasznosak mikor több paraméter szerepel alapértelmezett értékkel és csak egyet akarunk specifikálni, mikor meghívjuk.

static void MyMethod(string child1 = "Liam", string child2 = "Jenny", string child3 = "John")
{
Console.WriteLine(child3);
}
static void Main(string[] args)
{
MyMethod("child3");
}
// John
